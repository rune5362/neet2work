### DB л§Ҳмқҙк·ёл Ҳмқҙм…ҳ

PostgreSQL мҠӨнӮӨл§Ҳ кіөмң лҠ” Prisma MigrateлҘј мӮ¬мҡ©н•©лӢҲлӢӨ.

Prisma migrationмқҖ м•„лһҳ кө¬мЎ°лЎң кҙҖлҰ¬н•©лӢҲлӢӨ.
```

    в”ңв”Җ schema.prisma
    в”ңв”Җ seed.ts

       в”ңв”Җ migration_lock.toml
       в””в”Җ 00000000000000_init/

```

`00000000000000_init/migration.sql`мқҖ мқҳлҜё м—ҶлҠ” мҳҲмӢң нҢҢмқјмқҙ м•„лӢҲлқј, нҳ„мһ¬ н”„лЎңм қнҠёмқҳ мҙҲкё° DB мҠӨнӮӨл§ҲлҘј л§Ңл“ңлҠ” мӢӨм ң migration нҢҢмқјмһ…лӢҲлӢӨ. мқҙ нҢҢмқјмқҖ `job_postings`, `resume_analyses`, `AnalysisMode` enum, мқёлҚұмҠӨ, мҷёлһҳнӮӨ кҙҖкі„лҘј мғқм„ұн•©лӢҲлӢӨ.

л§Ҳмқҙк·ёл Ҳмқҙм…ҳ мӢӨн–ү мҲңм„ңлҠ” `prisma/migrations/` м•„лһҳмқҳ нҸҙлҚ”лӘ… мҲңм„ңлЎң кІ°м •лҗ©лӢҲлӢӨ.
```
 
    1. 00000000000000_init/migration.sql
    2. 20260513153000_add_users/migration.sql
    3. 20260514101000_add_resume_history/migration.sql

```

мӢӨм ң мһ‘м—…м—җм„ңлҠ” Prismaк°Җ ліҙнҶө `20260513..._name` нҳ•мӢқмқҳ migration нҸҙлҚ”лҘј мһҗлҸҷ мғқм„ұн•©лӢҲлӢӨ.

мғҲ мҠӨнӮӨл§Ҳ ліҖкІҪмқ„ л§Ңл“Ө л•ҢлҠ” м•„лһҳ лӘ…л №мқ„ мӮ¬мҡ©н•©лӢҲлӢӨ.
```
 
    npm run db:migrate -w apps/backend -- --name add_feature_name

```

лҸҷлЈҢк°Җ л§Ңл“  migrationмқ„ л°ӣмқҖ л’Өм—җлҠ” м•„лһҳ мҲңм„ңлЎң л§һм¶ҘлӢҲлӢӨ.
```
 
    git pull
    npm run db:migrate
    npm run db:seed

```

кё°мЎҙ мҠӨнӮӨл§ҲлҘј нҒ¬кІҢ л°”кҫёкі  мӢ¶мқ„ л•ҢлҠ” мғҒнҷ©м—җ л”°лқј лӢӨлҘҙкІҢ мІҳлҰ¬н•©лӢҲлӢӨ.

  * нҢҖмӣҗл“Өмқҙ м•„м§Ғ migrationмқ„ м Ғмҡ©н•ҳкё° м „: кё°мЎҙ мҙҲкё° migrationмқ„ лӢӨмӢң л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ.
  * нҢҖмӣҗл“Өмқҙ мқҙлҜё migrationмқ„ м Ғмҡ©н•ң л’Ө: кё°мЎҙ migrationмқҖ мҲҳм •н•ҳм§Җ м•Ҡкі  мғҲ migrationмқ„ м¶”к°Җн•©лӢҲлӢӨ.
  * лЎңм»¬ к°ңл°ң DB лҚ°мқҙн„°к°Җ н•„мҡ” м—Ҷмқ„ л•Ң: `npm run db:reset`мңјлЎң лЎңм»¬ DBлҘј мҙҲкё°нҷ”н•ң л’Ө seedлҘј лӢӨмӢң л„ЈмҠөлӢҲлӢӨ.



мЈјмқҳмӮ¬н•ӯ:

  * мқҙлҜё кіөмң лҗң migration нҢҢмқјмқҖ лҗҳлҸ„лЎқ мҲҳм •н•ҳкұ°лӮҳ мӮӯм ңн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.
  * кіөмң  нӣ„ DB ліҖкІҪмқҖ мғҲ migrationмңјлЎң лҲ„м Ғн•©лӢҲлӢӨ.
  * мӢӨм ң лЎңм»¬ DB лҚ°мқҙн„°лҠ” GitмңјлЎң кіөмң н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.
  * кіөмң н•  мғҳн”Ң лҚ°мқҙн„°лҠ” `apps/backend/prisma/seed.ts`м—җ л°ҳмҳҒн•©лӢҲлӢӨ.
  * м„ңлІ„ мӢӨн–үмқҖ DB м—ҶмқҙлҸ„ Mock fallbackмңјлЎң к°ҖлҠҘн•ҳм§Җл§Ң, `db:migrate`, `db:seed`лҠ” мӢӨм ң DB м—°кІ°мқҙ н•„мҡ”н•©лӢҲлӢӨ.
  * `npm run setup`, `npm run db:generate`, `npm run dev`лҠ” DB м—ҶмқҙлҸ„ мӢӨн–ү к°ҖлҠҘн•ҳлҸ„лЎқ мң м§Җн•©лӢҲлӢӨ.
