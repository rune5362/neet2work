# 최종 채용 사이트 수집 대상

확정일: 2026-05-13

## 목적

Neet2Work 채용공고 수집기에서 실제로 사용할 한국/일본 채용 사이트를 최종 정리한다. 기준은 브라우저 자동화 없이 `HTTP 요청 + HTML 파싱`으로 목록과 상세 공고를 수집할 수 있는지다.

JS 렌더링이 필수인 사이트, 봇 검증에 막히는 사이트, 로그인 의존도가 큰 사이트, 공식 API 신청이 필요한 사이트는 최종 수집 대상에서 제외한다.

## 최종 결론

| 구분 | 최종 사용 사이트 | 비고 |
| --- | --- | --- |
| 한국 | 사람인, 잡코리아, 캐치, 링커리어, 잡플래닛, Indeed Korea | Indeed는 보조/소량 수집 |
| 일본 | 마이나비 전직, doda, 리쿠나비 NEXT, Daijob, CareerCross, Green | 일본 종합 3개 + 글로벌/IT 특화 3개 |

## 구현 메모

- 1차 구현은 `HTTP 요청 + HTML 파싱`으로 시작한다.
- JS 렌더링 전용 Playwright 수집기는 이번 최종 대상에서는 제외한다.
- 사이트별 selector가 깨질 수 있으므로 `source`, `sourceJobId`, `sourceUrl`, `rawText`를 반드시 저장한다.
- 일본 사이트는 원문 언어가 일본어/영어이므로 `country`, `language`, `currency`, `languageRequirements`, `visaSupport` 필드를 함께 둔다.
- Indeed Korea는 캡차와 중복 리스크가 있으므로 대량 수집원이 아니라 보조 수집원으로 제한한다.
